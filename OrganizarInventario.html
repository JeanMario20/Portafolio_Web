Organizar Inventario

Roles:

BackEnd, FrontEnd, dataBase

Tecnologias:

Python, Flask, sqlAlchemy, Vuejs, javascript, Css, HTML

Meta:

Crear una aplicacion web donde el usuario pueda almacenar y filtrar el inventario de su tienda en tiempo real sin cargas de pantallas.

Objetivos personales

-Entender la estructura del desarrollo de aplicaciones con Flask.

-Entender la estructura del trabajo de la herramienta frontEnd vuejs

-Mandar y recibir peticiones mediante api rest con HTTPS.

-Crear las peticiones en frontEnd para procesarlos en backend y que de ahi tengan una respuesta a modificar la base de datos.

-La pagina no tenga carga de pantalla o buffers de carga

Muestra previa de la aplicacion lo que el usuario ve y tiene a su disposicion.

Botones de flechas para aumentar y disminuir el num de inventario

Boton rojo para eliminar un registro del inventario

Formulario para agregar un nuevo elemento ya sea inventario o bodega
