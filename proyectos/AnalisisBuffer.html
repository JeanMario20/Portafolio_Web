Herramienta de Análisis Buffer

SIGSA

Roles:

Back-End, Front-End

Lenguajes:

JavaScript, CSS, HTML

Meta:

Crear una herramienta que visualice, en una interfaz gráfica, el recorrido de los ductos petroleros y detecte a su alrededor estructuras o elementos que puedan obstruir el ducto. También, identificar el área que abarcan dentro de esas estructuras y obtener un análisis de las coordenadas obtenidas.

Objetivos

- Desarrollar la herramienta como un widget dentro del contexto de la aplicación Web AppBuilder de ArcGIS.

- Trabajar con la API JavaScript de ArcGIS para los procesos y funciones geológicas.

- Comunicar los servicios almacenados de la empresa para que se reflejen en la herramienta (capas de estructuras, buffers).

- Desarrollar las diferentes acciones que tendrá el ducto para obtener los datos y coordenadas (funciones de obstrucción, interpolación, área, aproximación, comportamiento del buffer).

- Administrar e integrar capas externas proporcionadas por la empresa para que interactúen con las funciones del ducto, así como gestionar los datos que estas devuelvan como respuesta, para mostrarlos en el informe final.

- Procesar las coordenadas e información obtenidas para el informe final.

Factibilidad

Muestra previa de la aplicación: lo que el usuario puede ver y utilizar.

Tipo de análisis: muestra los tipos de análisis para obstrucción (Factibilidad) y análisis de la abarcanza del ducto en la zona (Afectación).

Elige tu dibujo: ducto (línea recta), pozo para análisis de áreas (punto), zonas de afectación (figura polígono).

Cargar archivo: permite cargar un archivo de un dibujo ya hecho en formato KML.

Tipo de buffer: diseño del buffer plano o redondo.

Zona de buffer: determinar los lados en los que existirá el buffer.

Medidas: asignar las distancias que abarcará el buffer.

Ejecutar análisis: comenzar el análisis.

Informe: descargar la información en archivo PDF o Excel.

Borrar: eliminar todo el proceso.
