Organizar Inventario

Roles:

BackEnd, FrontEnd, DataBase

Tecnologias:

Python, Flask, SQLAlchemy, Vuejs, JavaScript, CSS, HTML

Meta:

Crear una aplicacion web donde el usuario pueda almacenar y filtrar el inventario de su tienda en tiempo real sin cargas de pantallas.

Objetivos personales

- Entender la estructura del desarrollo de aplicaciones con Flask.

- Comprender la estructura del trabajo de la herramienta frontend Vue.js.

- Mandar y recibir peticiones mediante API REST con HTTPS.

- Crear las peticiones en el frontend para procesarlas en el backend y, desde ahí, generar una respuesta que modifique la base de datos.

- Lograr que la página no tenga tiempos de carga visibles o buffers.

Muestra previa de la aplicación: lo que el usuario puede ver y utilizar.

Botones de flechas para aumentar y disminuir el número de inventario.

Botón rojo para eliminar un registro del inventario.

Formulario para agregar un nuevo elemento, ya sea inventario o bodega.

El formulario aparece cuando se oprime el botón de "agregar".

Evita que se agreguen elementos repetidos.

La barra de búsqueda encuentra el ítem específico.

Si no se encuentra el elemento, se notificará al usuario.

Bodega para guardar otros tipos de elementos.

El formulario aparece cuando se oprime el botón de "agregar".
